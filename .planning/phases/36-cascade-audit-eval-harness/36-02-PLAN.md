<task type="execute">
<action>
Create evals/cascade_audit/run_audit.py CLI orchestrator:

1. Define CLI argument parser:
   - --round: round number (0, 1, or 2)
   - --callsites: comma-separated list of callsites
   - --providers: comma-separated list of providers
   - --db-path: path to production DB
2. Implement main function:
   - Parse CLI arguments
   - Validate round number (must be 0, 1, or 2)
   - Load config from config.yaml
   - Initialize CorpusLoader
   - For Round 0: call load_round_0(n=3 per callsite)
   - For Round 1: call load_round_1()
   - For each (callsite, provider) pair:
     - Load adapter
     - Exercise adapter on corpus
     - Compute metrics
     - Write artifact atomically to artifacts/round_N/{callsite}_{provider}.json
   - Generate report using report.py
3. Implement atomic artifact write helper:
   - Write to temp file (.json.tmp)
   - Rename to final path (atomic on Windows)
4. Add provenance block to each artifact:
   - provider_config snapshot
   - model_versions
   - harness_commit_sha (from git)
   - sample_seed
   - scheduler_pause_status
</action>
