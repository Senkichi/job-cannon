# Plan 04: Per-Callsite Adapters

<objective>
Implement 6 per-callsite adapters for the non-scoring callsites (parse_structured_fields, find_careers_url, extract_jobs, description_reformat, company_research, ai_nav_discovery) following the TaskAdapter protocol.
</objective>

<action>
Create evals/cascade_audit/adapters/__init__.py with TaskAdapter protocol:

1. Import Protocol from typing
2. Define TaskAdapter Protocol:
   - sample(n: int, conn) -> list[dict]
   - exercise(row: dict, provider: str, config: dict, conn) -> dict
   - score(gold: dict, candidate: dict) -> dict
3. Add docstring explaining the protocol purpose
4. Export TaskAdapter in __all__
</action>

<task type="execute">
<action>
Create evals/cascade_audit/adapters/parse_structured_fields_adapter.py:

1. Define ParseStructuredFieldsAdapter class implementing TaskAdapter
2. sample method:
   - Query jobs table: SELECT dedup_key, jd_full FROM jobs WHERE jd_full IS NOT NULL AND LENGTH(jd_full) > 400 ORDER BY RANDOM() LIMIT ?
   - Return list of dicts
3. exercise method:
   - Import parse_structured_fields from job_finder.web.enrichment_tiers
   - Call parse_structured_fields(jd_full=row["jd_full"], provider=provider, config=config, conn=conn)
   - Return result dict
4. score method:
   - Compute schema_valid: isinstance(candidate, dict) and all(k in candidate for k in gold)
   - Compute salary_mae if both salary_min populated
   - Compute location_match: gold.get("location") == candidate.get("location")
   - Compute hallucination_count: len(set(candidate.keys()) - set(gold.keys()))
   - Return metrics dict
</action>

- evals/cascade_audit/adapters/parse_structured_fields_adapter.py exists

</acceptance_criteria>
</task>
